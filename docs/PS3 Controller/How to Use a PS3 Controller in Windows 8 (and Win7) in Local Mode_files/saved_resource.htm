            How to use Motion Joy – Guide for built in Bluetooth,
            Bluetooth dongle and USB (Windows 8 and Below(

            Through USB-

            1.
            Download this motion joy and offline mode first(Top
            of this article), then install it

            2.
            Copy the local mode file to the C:\Program
            Files\MotioninJoy\ds3

            3.
            Turn off your Bluetooth first or do not plug in
            your Bluetooth first, then make sure you connect PS3 controller on computer and
            open motion joy application, then select local→install (Let your computer install
            driver as PS3 controller first, if you offline it could be failed to install
            driver, let it be)

            4.
            Tick your driver port and load driver, it must
            be successfull. If you failed, read at the bottom of this tutorial

            5.
            Done

            Through Bluetooth-

            1.
            After done the first part (USB), turn on your
            built in Bluetooth or Bluetooth dongle

            2.
            Tick your driver port and load driver, it must
            be successful. If you failed read at the bottom of this tutorial

            3.
            Done.

            If you failed in USB -

            1.
            Reboot your computer and go to safe mode by pressing F8. Then choose disable driver signature. (If you are
            in windows 8 just like me, you could go to enter password screen and hold down
            shift button while select restart, if not you could learn on google by type “how
            to disable driver signature”)

            2.
            Try again
            the step (Through USB) and it should work now when your load driver

            3.
            Done

            If
            you failed in Built in Bluetooth or
            Bluetooth Dongle

            1.
            Then, after reboot go to your C:\Program
            Files\MotioninJoy\ds3\drivers\ and open the MijXinput.inf

            2.
            Go to your windows section(if you don’t know
            your windows section, read at the bottom of this tutorial) neither (NTx86,NTamd64,NTx86.6.0,NTamd64.6.0)
            and go to ;BLUETOOTH DONGLE at little bit down it.

            3.
            Here for an example,

            (PIC 1.jpg)

            4.
            Copy your hardware
            id (you can get it by open motion
            joy and go to local→install, it should show your hardware id that failed to load) to the bottom of ;BLUETOOTH DONGLE ,

            Ex. %MIJ.DeviceName%=CC_Install,
            (Your hardware id) – Make sure it bottom of the line before another windows
            section and have space between it just like below)

            (PIC 2.jpg)

            5.
            Save the .inf when close the notepad.

            6.
            If you failed to save, copy first MijXinput.inf to your desktop and edit
            it there, then replace it with the same file.

            7.
            After successfully edit the file, open your
            motion joy and repeat step (Through Bluetooth) and it should work now when you
            load the driver

            If you don’t know your windows section, check it by :

            1.
            Open motion joy application, go to local→install and select Install All

            2.
            It should show some error with NT(windows section)

            I learn from others too, so no need to thank
            me…Enjoy using your PS3 Controller…HAHA
